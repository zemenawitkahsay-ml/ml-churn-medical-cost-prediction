# Machine Learning Project: Customer Churn & Medical Cost Prediction

## Overview
This project demonstrates the application of machine learning techniques to solve two real-world predictive problems:

1. **Customer Churn Prediction (Classification)** – Predict whether a customer is likely to leave a service.
2. **Medical Insurance Cost Prediction (Regression)** – Estimate healthcare insurance charges based on individual attributes.

The project implements multiple machine learning models and compares their performance using appropriate evaluation metrics.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Datasets

### Customer Churn Dataset
The churn dataset contains customer demographic and behavioral information used to predict whether a customer will churn.

Typical features include:
- Gender
- Tenure
- Contract type
- Monthly charges
- Total charges
- Payment method

Target variable:
- **Churn (Yes / No)**

### Medical Insurance Dataset
This dataset contains information used to predict medical insurance costs.

Features include:
- Age
- Sex
- BMI
- Number of children
- Smoking status
- Region

Target variable:
- **Insurance Charges**

---

## Machine Learning Models

### Classification Models (Churn Prediction)

The following algorithms were used to classify customer churn:

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest
- AdaBoost
- XGBoost

These models were trained and compared to determine the most effective algorithm for predicting churn.

---

### Regression Models (Medical Cost Prediction)

Several regression models were applied to predict medical insurance charges:

- Linear Regression
- K-Nearest Neighbors Regressor
- Support Vector Regressor (SVR)
- Random Forest Regressor
- AdaBoost Regressor
- XGBoost Regressor

---

## Machine Learning Workflow

The project follows a standard machine learning pipeline:

1. Data loading
2. Data cleaning and preprocessing
3. Exploratory data analysis (EDA)
4. Feature engineering
5. Train-test split
6. Model training
7. Model evaluation
8. Model comparison

---

## Evaluation Metrics

### Classification Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

### Regression Metrics
- Root Mean Squared Error (RMSE)
- R² Score

---
## Project Structure
```
ml-churn-medical-cost-prediction
│
├── churn_medical_cost_prediction.ipynb
├── Medical_insurance.csv
├── telco_customer_churn.csv
├── README.md
├── LICENSE

```

---

## Key Insights
- Multiple machine learning models were compared for both classification and regression tasks.
- **Logistic Regression** achieved the best performance for **customer churn classification**.
- **XGBoost Regressor** achieved the best performance for **medical cost prediction**.
- Overall, the project demonstrates the importance of model comparison and evaluation when building predictive systems.
